This project covers the 3D modeling, 3D printing prototyping and functional performance verification of a servo-driven movable tire base for truck scale models. All core mechanical components, including the chassis mounting bracket, swing connecting link, steering crank arm, wheel fixing support and gear transmission assembly, are designed with 3D modeling software.
After model derivation, 3D printing fabrication and physical hardware assembly, multiple practical tests are conducted to evaluate dimensional machining error, linkage motion smoothness and wheel steering reliability. Combined with the test data recorded in experimental records, structural design and assembly tolerance of the 3D model are iteratively optimized. The ultimate goal of this project is to develop a mechanical base that enables precise controllable wheel steering motion actuated by miniature servo motors.
