This project implements the design, 3D printing prototyping and functional validation of a servo-controlled movable tire base for truck models. All structural parts such as chassis brackets, steering linkages, crank arms and gear transmission components are modeled in 3D software.
After 3D printing and physical assembly, tests on dimensional accuracy, linkage fluency and steering performance are completed. The 3D structural design and assembly tolerances are optimized according to experimental test data. This work finally achieves a mechanical platform that supports servo-driven controllable wheel steering.
