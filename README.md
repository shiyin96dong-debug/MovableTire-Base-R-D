# MovableTire-Base-R-D
This project focuses on 3D modeling, prototype printing and functional verification of a servo-controlled movable tire base for truck model. All core structural parts including chassis bracket, steering rocker arm, tie rod and wheel mount are designed via 3D modelling software. After model export, 3D printing and physical assembly, practical tests on dimensional error, linkage smoothness and steering performance are carried out, and the 3D model structure & tolerance are optimized based on test data recorded in Material Result. The final target is to complete a mechanical base that can realize controllable wheel steering driven by micro servo.
